📰 Fake News Detection Web App

A Machine Learning based web application that classifies news articles as Fake or Real using Natural Language Processing (TF-IDF) and a trained classification model. The model is integrated into a Streamlit web interface for real-time predictions.

🚀 Features

✅ Real-time fake news prediction

✅ TF-IDF text vectorization

✅ Trained ML classification model

✅ Interactive Streamlit UI

✅ Cloud deployment

🛠️ Tech Stack

Python

Pandas

Scikit-learn

Streamlit
